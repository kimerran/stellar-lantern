Install the Beta in 5 steps

Lantern isn't on the Chrome Web Store yet. You can load it directly in Developer Mode — it takes about a minute.

1

Download & unzip

Grab the extension package from our Drive and unzip it to a folder you'll remember.

2

Open Chrome extensions

Go to chrome://extensions in your address bar.

3

Enable Developer Mode

Flip the Developer mode switch in the top-right corner.

4

Load unpacked

Click Load unpacked and select the unzipped dist folder.

5

Pin & open

Pin Lantern to your toolbar, click the lantern, and create or import a wallet.
